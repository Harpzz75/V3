These could be the very commonly used .jpg also filetypes such as .png, .gif, .html, .css, .js, .mpeg, etc etc.

Different file types do different things, .jpg, .gif, .png are image file formats. HTML, CSS, JavaScript etc are web languages. Usually websites and associated images and content would be hosted on a server, most often offsite somewhere. The web page information is referenced during the web journey and called upon and rendered through a browser. An important consideration for the designer and developer during the construction of the site is how the website will handle images. As a web designer these are some of the most important things to remember when creating web image content.
